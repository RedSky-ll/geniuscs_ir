# جنیوس سی‌اس ایران | GeniusCS IR

وب‌سایت رسمی آژانس دیجیتال **جنیوس سی‌اس (GeniusCS)** در ایران؛ نماینده رسمی آژانس کانادایی `GeniusCS.ca`. این وب‌سایت یک پلتفرم مدرن، مینی‌مال و با تمرکز بر تجربه کاربری (UX) است که تخصص‌های این آژانس را در زمینه‌های طراحی وب، اپلیکیشن، امنیت اطلاعات و هوش مصنوعی به نمایش می‌گذارد.

## ویژگی‌های برجسته | Key Features

- **نمایندگی رسمی:** طراحی شده برای نمایش هویت آژانس مادر در کانادا با استانداردهای بین‌المللی.
- **تکنولوژی مدرن:** توسعه داده شده با **Next.js 16** و **Tailwind CSS v4**.
- **انیمیشن‌های پیشرفته:** استفاده گسترده از **Motion (Framer Motion)** برای انیمیشن‌های مبتنی بر اسکرول (Scroll-driven) مشابه سایت `motion.dev`.
- **نمایش گرافیکی محصولات:** ویترین عمودی پروژه‌ها در قاب‌های شبیه‌سازی شده لپ‌تاپ و موبایل.
- **تایپوگرافی حرفه‌ای:** استفاده از فونت **یکان بخ (Yekan Bakh)** برای ایجاد تجربه بصری برتر در زبان فارسی.
- **طراحی ریسپانسیو:** بهینه‌سازی کامل برای موبایل با منوی همبرگری انیمیشنی.
- **تمرکز بر تماس مستقیم:** سیستم تماس سریع و مستقیم تلفنی برای مشاوره پروژه‌ها.

## تکنولوژی‌های مورد استفاده | Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion (Motion.dev)
- **Icons:** Lucide React
- **UI Components:** Shadcn UI

## نصب و راه‌اندازی | Installation

برای اجرای پروژه در محیط محلی:

```bash
# نصب وابستگی‌ها
npm install

# اجرای سرور توسعه
npm run dev
```

سپس آدرس `http://localhost:3000` را در مرورگر خود باز کنید.
